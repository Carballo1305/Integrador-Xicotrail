# Integrador-Xicotrail
Base de Datos – Xicotrail
Base de datos relacional creada como proyecto académico para el control básico del transporte público.

Incluye tablas para:
Usuarios
Operadores
Unidades
Rutas
Relación entre operadores, unidades y rutas
Proyecto enfocado en el aprendizaje de fundamentos de bases de datos y SQL.
